# Wide Brain Technologies — Minimalist Website

## Run it
- Open `index.html` in your browser (double-click it).

## Customize
- **Brand + copy**: `index.html`
- **Colors / styling**: `styles.css` (see `:root` color variables at the top)
- **Interactions** (mobile menu, counters, copy email, toast): `script.js`

## Notes
- The “Featured videos” are placeholders—swap the `href`, titles, and thumbnail blocks with your real YouTube links and images when ready.
